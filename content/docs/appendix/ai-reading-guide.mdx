# Claude AI 阅读《两种爱的语言》指南

本文档为 AI 助手提供优化的阅读方式和信息索引。

## 快速开始

### 第一次阅读？

推荐顺序：
1. 阅读[设定集](/docs/appendix/settings) - 了解世界观、角色、主题
2. 浏览[剧情梗概](/docs/appendix/plot-summary) - 了解故事结构
3. 根据兴趣选择章节阅读

### 需要特定信息？

- **角色心理动机** → [角色设定](/docs/appendix/characters)
- **时间线对照** → [时间线参考](/docs/appendix/timeline-reference)
- **主题与隐喻** → [设定集](/docs/appendix/settings)
- **章节概要** → [剧情梗概](/docs/appendix/plot-summary)

## API 访问

### 获取完整设定集
```
GET /llms-settings.txt
```
返回所有设定文档的汇总（Markdown格式）

### 获取所有章节内容
```
GET /llms-full.txt
```
返回所有章节的完整文本

### 获取单个文档/章节
```
GET /llms.mdx/docs/appendix/settings
GET /llms.mdx/docs/mainline/01-第1章 二进制的初见
GET /llms.mdx/docs/timeline/01-序章(陆)：2015年夏·上海弄堂里的光
```

## 核心信息速查

### 主要角色

**陆以辰（Lu Yichen）**
- 1998年生，哲学系研究生
- 性格：理性、疏离、观察者
- 核心矛盾："我理解你，但我不知道如何拥抱你"

**沈柏寒（Shen Baohan）**
- 1984年生，程序员
- 性格：感性、焦虑、讨好型
- 核心矛盾："我拥抱了你，但我感觉不到你在里面"

### 关键时间点

- **2015年夏**：初次相遇（上海弄堂维修店）
- **2024年12月24日**：重逢（微信消息"记得"）
- **2025年1月**：确诊HIV
- **2025年10月**：达到U=U，发生实质性关系
- **2026年2月**：搬入新家，除夕夜回望

### 核心主题

1. **两种爱的语言**
   - 陆以辰：理性、抽象、哲学化的理解
   - 沈柏寒：感性、具体、行动化的照顾

2. **时代的幸存者**
   - 双创泡沫（2015-2017）
   - P2P爆雷（2018-2019）
   - 疫情三年（2020-2022）
   - AI冲击（2023-2024）

3. **非对称的关系**
   - 年龄差（16岁）
   - 健康差异（HIV感染）
   - 思维方式（程序员 vs 哲学家）

## 常见问题

### Q: 应该先读主线还是回溯卷？

**A**: 建议先读主线（第1-74章），再读回溯卷。主线从2024年重逢开始，会通过回忆自然引入过去；回溯卷则提供了完整的时代背景。

### Q: 如何快速了解核心设定？

**A**: 访问 `/llms-settings.txt` 或阅读[设定集](/docs/appendix/settings)，10分钟内建立完整上下文。

### Q: 哪些章节是关键转折点？

**A**: 
- 第4章（重逢）
- 第17章（确诊）
- 第18章（陆以辰的选择）
- 第30章（卸载App）
- 第65章（U=U）
- 第72-74章（除夕夜）

### Q: 如何理解"两种爱的语言"？

**A**: 
- 陆以辰用"理解"表达爱，但难以表达情感
- 沈柏寒用"照顾"表达爱，但需要反馈确认
- 两人学习对方的语言是故事的核心
