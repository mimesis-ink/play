# 配角卷

围绕在主角周围的配角们，每个人都有自己的故事和挣扎。

## 章节规划

### 配角卷·第一章：周蔓的视角
倒叙她在婚姻中的绝望，以及离婚后的重生。当她得知沈柏寒确诊，她的复杂心情。

### 配角卷·第二章：林骁的告白
他的完整故事：艺术梦的破碎，肉体交易的沉沦。他和沈柏寒的几次相遇，以及最后的诀别。

### 配角卷·第三章：老张的黄昏
IT行业十年变迁，通过老张的职业生涯展现。他被裁员，转型，与沈柏寒的对话。

### 配角卷·第四章：陆母的笔记
第一份心理观察报告，冷静解剖陆以辰。

### 配角卷·第五章：医院众生相
- 老李（隐瞒者）
- 小K（年轻的绝望者）
- 老王（NGO行动者）
三个病友的完整故事线。

### 配角卷·第六章：疾控中心的24小时
跟随一个疾控医生的视角，看到流调、登记、咨询。展现制度的冷漠与个体的温暖。

### 配角卷·第七章：导师林教授的两难
他如何看待陆以辰的选择。学术圈的现实与理想主义的冲突。
