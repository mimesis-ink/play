【2026年7月27日，下午五点】

　　比赛结束了。 

　　沈柏寒、陆以辰和小宇走出科技馆，朝着停车场走去。

　　小宇抱着那个比他脸还大的奖杯，走得很慢，因为他在看脚下的每一块地砖——数着它们的数量，计算着铺设的规律。

　　"一共564块。"他突然说，"8的倍数排列，边缘用了半块砖补齐。"

　　陆以辰笑了："你连地砖都不放过？"

　　"因为它们很有规律。"小宇认真地说，"规律的东西让我觉得舒服。"

　　他们走到停车场，沈柏寒按了遥控钥匙。

　　就在这时，一个声音从身后传来：

　　"家长，请等一下。"

　　沈柏寒转过身，看见一个穿着休闲装的中年男人正朝他们走来。

　　五十岁左右，头发有些乱，T恤上印着一行英文："Code is Poetry"（代码即诗歌），牛仔裤膝盖处有些磨损，脚上穿着一双旧运动鞋。

　　不像评委，更像个程序员。

　　但沈柏寒记得他——刚才就是这个人，在台上给了小宇最高的评价。

　　"您好。"沈柏寒说，"有什么事吗？"

　　"你好，我叫叶青，"男人伸出手，"是刚才的评委之一。"

　　"我想问一下，那个'噪音红绿灯'的代码，真的是他自己写的？"

　　沈柏寒握了握手，点头："是的。有什么问题吗？"

　　"没有问题。"叶青说，"恰恰相反——逻辑非常漂亮。"

　　"特别是那个防抖动算法，用计时器判断噪音是否持续超标，避免误报。这个处理得很老练，不像是一个五岁孩子能想到的。"

　　沈柏寒愣了一下，然后笑了："五岁？他已经六岁了。"

　　"六岁也一样。"叶青说，"大部分六岁的孩子，连变量和函数都分不清。但他的代码，有工程思维。"

　　"因为我教过他。"沈柏寒说。

　　"你是做技术的？"

　　"嗯，程序员。"

　　叶青眼睛亮了起来："难怪。那孩子的焊接手法呢？也是你教的？"

　　"是。"

　　"很好。"叶青说，"我看过他的电路板，焊点虽然大小不一，但没有虚焊，也没有短路。说明他很细心。"

　　沈柏寒有些疑惑："叶先生，您找我们，是想……？"

　　"哦，抱歉。"叶青笑了，"我还没自我介绍。我除了是评委，还有另一个身份——"

　　他从口袋里掏出一张名片，递给沈柏寒。

　　名片很简洁，只有几行字：

　　**叶青**
　　**上海源码实验学校 创始人/校长**
　　**www.sourcecodeschool.edu.cn**

　　沈柏寒看着这张名片，愣住了。

　　"源码实验学校？"陆以辰在旁边念道。

　　"对。"叶青说，"一所有点特别的小学。"

　　"我们专门招收像他这样的孩子——"他指了指小宇，"有点与众不同，但很有天赋的孩子。"

　　"如果你们还没决定好让他去哪上小学，下周一，带他来我们学校看看。"

　　沈柏寒和陆以辰对视了一眼。

　　"您的意思是……"沈柏寒小心翼翼地问，"您想招他入学？"

　　"如果你们愿意的话。"叶青说，"当然，得先参观一下学校，看看环境是否合适。"

　　"我们的学校有点……乱。"他笑了笑，"可能和你想象中的学校不太一样。"

　　"但我觉得，"他看着小宇，"他会喜欢的。"

　　小宇正蹲在地上，盯着停车场的排水口。

　　"这个排水口的栅格间距是2.1厘米。"他说，"刚好能卡住直径大于2厘米的物体。设计得很合理。"

　　叶青听了，大笑起来："看，这就是我喜欢的孩子。"

　　"别人看排水口只是看排水口，他看到的是工程设计。"

　　"这种思维方式，太珍贵了。"

　　---

　　**【2026年8月3日，周一上午】**

　　上海源码实验学校。

　　这是一栋不太起眼的三层小楼，位于杨浦区一条安静的街道上。

　　外墙是灰白色的，没有华丽的装饰，也没有醒目的校名——只有门口一块木制的牌子，上面刻着：

　　**Source Code School**
　　**源码实验学校**
　　**让每个孩子成为自己的源代码**

　　沈柏寒看着这行字，心里涌起一种奇妙的共鸣。

　　"源代码"——每个程序员都知道这个词的分量。

　　它不是编译后的二进制，不是包装好的软件，而是最原始的、可读的、可修改的代码本身。

　　把孩子比作源代码……

　　这个隐喻，太准确了。

　　他们走进学校大门。

　　迎接他们的，是叶青。

　　今天他穿着一件黑色T恤，上面印着一行绿色的代码：

　　```
　　while(alive) {
　　learn();
　　}
　　```

　　"欢迎。"叶青笑着说，"来，我带你们参观一下。"

　　他推开一楼的门。

　　沈柏寒和陆以辰走进去，然后愣住了。

　　这……这真的是学校吗？

　　---

　　**【教室：没有围墙的学习空间】**

　　一楼是一个开放式的大空间，大约两百平米。

　　没有传统意义上的"教室"。

　　没有整齐划一的课桌椅。

　　也没有黑板和讲台。

　　取而代之的，是：

　　**区域一：创客工坊**

　　几张大工作台，上面摆满了工具——烙铁、螺丝刀、万用表、3D打印机、激光切割机。

　　几个孩子正在那里组装什么东西，零件散落一地。

　　有个七八岁的男孩，正在用螺丝刀拆一个旧收音机。

　　"小心点，别弄坏了。"旁边的女孩说。

　　"我就是要拆开看看里面是什么。"男孩说，"不拆怎么知道它是怎么工作的？"

　　**区域二：阅读角**

　　一整面墙的书架，从地板延伸到天花板。

　　各种各样的书——科学、文学、历史、艺术、编程、哲学。

　　几个孩子躺在懒人沙发上看书，姿势千奇百怪。

　　有个女孩趴在地毯上，把书摊开，一边看一边在笔记本上画思维导图。

　　**区域三：编程区**

　　十几台电脑，没有固定的座位。

　　几个孩子在那里敲代码，屏幕上滚动着彩色的文本。

　　有个看起来才八岁的男孩，正在用Scratch做一个小游戏。

　　旁边的女孩在帮他调试："这里的碰撞检测逻辑不对，你看……"

　　**区域四：艺术墙**

　　一整面墙被涂成了黑板墙。

　　上面画满了涂鸦——有数学公式，有卡通人物，有一首诗，还有一个复杂的流程图。

　　字迹大小不一，颜色五颜六色。

　　很乱，但很有生命力。

　　沈柏寒看着这一切，脑子里只有一个念头：

　　这不像学校。

　　更像……Google的办公室，或者某个创业公司的联合办公空间。

　　"我知道你在想什么。"叶青笑着说，"这不像传统的学校。"

　　"但在我看来，传统的学校有个问题——它把孩子当成工厂里的零件，要求他们整齐划一，按照同一个模具生产出来。"

　　"而我想做的，是把这里变成一个开源社区。"

　　"每个孩子都是一段独特的源代码，有自己的语法，自己的逻辑。"

　　"我们不要求他们都编译成同一个程序，我们只是提供一个环境，让他们能够自由地运行，自由地debug，自由地迭代。"

　　陆以辰听着，眼睛越来越亮。

　　这个教育理念，和他在哲学系学到的那些理论不谋而合——

　　海德格尔说的"让存在者成为它自己"。

　　萨特说的"存在先于本质"。

　　而叶青，用最朴素的语言，在实践着这些理论。

　　小宇站在门口,看着这个空间，没有说话。

　　但沈柏寒注意到，他的眼睛在发光。

　　---

　　**【第一次接触：齿轮的对话】**

　　"去看看吧。"沈柏寒轻轻推了推小宇。

　　小宇犹豫了一下，然后慢慢走进创客工坊区域。

　　那里有三个孩子，正在拼一个乐高机器人。

　　看起来是个四驱车，但齿轮系统卡住了，怎么都转不动。

　　"肯定是这里的问题。"一个男孩说，指着底盘。

　　"不对，我觉得是电机坏了。"女孩说。

　　"都不是。"第三个孩子说，"是齿轮装反了。"

　　小宇站在旁边，看了一会儿。

　　然后他走过去，指着其中一个齿轮："这个齿轮的方向是反的。"

　　三个孩子抬起头，看着他。

　　"你是谁？"男孩问。

　　"我叫小宇。"

　　"你怎么知道齿轮反了？"女孩问。

　　"因为……"小宇想了想该怎么解释，"因为这两个齿轮的旋转方向应该相反才能咬合。现在它们都是顺时针，所以会卡住。"

　　"把这个齿轮翻过来，就能转了。"

　　三个孩子对视了一眼。

　　"试试看。"女孩说。

　　男孩按照小宇说的，把齿轮拆下来，翻了个面，重新装上。

　　然后按下开关——

　　机器人动了！

　　四个轮子转了起来，车子在地板上跑了起来。

　　"哇！"三个孩子欢呼起来。

　　"你好厉害！"女孩看着小宇，眼睛亮晶晶的。

　　"你会修机器人？"男孩问。

　　"我会一点。"小宇说。

　　"那你能教我们吗？"女孩说，"我们想做一个能自己避障的车，但不知道该怎么编程。"

　　"可以。"小宇说，"需要一个超声波传感器，检测前方障碍物的距离。如果距离小于20厘米，就转向。"

　　"这个简单，就是一个if语句……"

　　他蹲下来，和三个孩子一起研究那个机器人。

　　沈柏寒站在不远处，看着这一幕。

　　他的眼眶有些湿润。

　　这是小宇第一次，主动和陌生的孩子交流。

　　不是被迫的，不是尴尬的，而是自然的。

　　因为他们在说同一种语言——

　　关于齿轮，关于传感器，关于代码。

　　这是小宇的母语。

　　而在这个学校里，有其他说这种语言的孩子。

　　陆以辰走到沈柏寒身边，轻轻握住他的手。

　　"你看。"他说，"他笑了。"

　　沈柏寒看过去。

　　小宇正在和那个女孩讨论超声波传感器的安装位置，脸上带着笑容。

　　那是真正的、放松的、发自内心的笑容。

　　不是在讨好谁，不是在假装融入，而是真正的快乐。

　　因为在这里，他不需要假装。

　　他可以做他自己。

　　---

　　**【校长办公室：昂贵的门票】**

　　半小时后，叶青把他们带到了二楼的办公室。

　　办公室也很"源码"——书架上堆满了书和杂物，墙上贴着各种便签纸，白板上写满了待办事项和教学计划。

　　"坐吧。"叶青指了指沙发。

　　沈柏寒和陆以辰坐下。

　　小宇还在楼下，和那几个孩子一起玩。

　　"我看得出来，他很喜欢这里。"叶青说。

　　"是的。"沈柏寒说，"我从来没见他这么放松过。"

　　"因为在这里，他是正常的。"叶青说，"你知道吗？我们学校有个孩子，七岁，能心算三位数的乘法，但不会系鞋带。"

　　"还有个孩子，八岁，能流利地说五种语言，但在人群中就会恐慌。"

　　"还有个孩子，六岁，能背出元素周期表，但写字歪歪扭扭。"

　　"他们在普通学校里，都是'问题儿童'。"

　　"但在这里，他们只是……不同而已。"

　　沈柏寒听着，心里涌起强烈的共鸣。

　　"叶校长，"陆以辰问，"您是怎么想到办这样一所学校的？"

　　叶青笑了笑，指了指墙上的一张照片。

　　照片里是个十几岁的男孩，戴着厚厚的眼镜，抱着一台电脑，笑得很腼腆。

　　"那是我儿子。"叶青说，"他小时候也是这样。对数字特别敏感，但社交能力很差。"

　　"普通学校说他是'自闭倾向'，建议我们带他去看医生。"

　　"但我知道，他不是病了，他只是……和别人不一样。"

　　"后来我把他转到了国外的一所特殊学校，专门收'天才怪胎'的那种。"

　　"他在那里找到了自己的位置，现在在硅谷做AI研究。"

　　"我想，如果国内也能有这样的学校，那该多好。"

　　"所以四年前，我辞职了，卖了房子，办了这所学校。"

　　沈柏寒听着，心里泛起一种复杂的情绪。

　　这个校长，和他太像了。

　　都是程序员出身，都有一个"不太一样"的孩子，都在用自己的方式，试图给这些孩子一个出口。

　　"叶校长，"沈柏寒说，"如果……如果小宇来这里上学，需要什么条件？"

　　叶青拿出一份招生简章，递给他们。

　　"我们不看户口，不看学区，也不需要考试。"他说，"我只看一件事——这个孩子是不是真的需要这里。"

　　"从刚才的观察来看，小宇很适合我们学校。"

　　"我们有专门的编程课，创客课，还有针对特殊需求孩子的社交训练。"

　　"但我必须提前告诉你们，"叶青停顿了一下，"我们的学费不便宜。"

　　沈柏寒翻开招生简章，看到了学费那一栏。

　　**年度学费：15万元**

　　**不含餐费、活动费、材料费**

　　他的手顿了一下。

　　15万。

　　这是他以前一年的工资。

　　现在，他做外包项目，一个月能赚一万多，好的时候两万。

　　一年下来，最多也就二十万。

　　扣掉房租、生活费、医疗费，能剩下多少？

　　而这个学费，就要占掉他大半的收入。

　　陆以辰也看到了那个数字，脸色有些凝重。

　　"我知道这个数字不小。"叶青说，"但私立学校的成本确实高——我们的师生比是1:5,每个老师都是我精心挑选的，工资也不能太低。"

　　"场地、设备、材料，都需要钱。"

　　"我们不是为了赚钱，但也不能亏本运营。"

　　沈柏寒盯着那个数字，脑子里在飞快地计算。

　　如果小宇上六年小学，就是90万。

　　他现在的积蓄，撑不过两年。

　　必须找到更稳定的收入来源。

　　或者……做点更大的事情。

　　他抬起头，看向窗外。

　　透过窗户，能看到楼下的院子。

　　小宇正在和那几个孩子一起,把机器人放在地上测试。

　　机器人走了几步，碰到了墙，自动转向，继续走。

　　孩子们欢呼起来。

　　小宇蹲在地上，在一个小本子上写着什么——应该是在记录测试数据。

　　那一刻，沈柏寒突然想起了叶青刚才说的话：

　　"他在那里找到了自己的位置。"

　　小宇，也在这里找到了自己的位置。

　　那些齿轮，那些传感器，那些代码——

　　这才是他的语言，他的世界。

　　如果能让他在这个世界里快乐地成长，90万算什么？

　　沈柏寒转回头，看着叶青，眼神坚定：

　　"名额我们要了。"

　　陆以辰愣了一下："柏寒……"

　　"钱不是问题。"沈柏寒说，"我会想办法。"

　　叶青看着他，眼神里有一种理解和欣赏。

　　"我明白你的顾虑。"他说，"这样吧，我们可以分期。每学期交一次，一年两次,每次7.5万。"

　　"另外,如果孩子在学校里有特别突出的表现,比如参加比赛获奖、发表作品之类的,我们会有奖学金。"

　　"这样压力会小一点。"

　　沈柏寒点点头："谢谢。"

　　"什么时候能办入学？"

　　"下周一开学。"叶青说，"暑期班已经在上了,正式的秋季学期是九月,但现在来也可以,先适应一下环境。"

　　"好。"沈柏寒说,"那我们下周一送他过来。"

　　---

　　**【回家的车上】**

　　下午四点，他们开车离开学校。

　　小宇坐在后座，抱着一本叶青送他的书——《Arduino编程指南》。

　　他翻开第一页，开始认真地读。

　　陆以辰坐在副驾驶，看着窗外,没有说话。

　　气氛有些沉默。

　　沈柏寒知道陆以辰在想什么。

　　钱。

　　15万的学费。

　　对一个还在读研的学生来说,这是一笔巨款。

　　虽然陆以辰没说,但沈柏寒知道他心里有愧疚——

　　觉得自己没有收入,帮不上忙,反而成了负担。

　　"以辰。"沈柏寒开口。

　　"嗯？"

　　"你在想什么？"

　　陆以辰沉默了一会儿："在想……那笔学费。"

　　"你的积蓄撑不了几年。"

　　"而我现在还在读书,没有收入,帮不上你。"

　　"也许……"他停顿了一下,"也许我应该先休学,找份工作。"

　　"不。"沈柏寒立刻说,"你必须读完博士。"

　　"但是……"

　　"没有但是。"沈柏寒握着方向盘,眼神很亮,"以辰,你知道我为什么坚持要让小宇上那所学校吗？"

　　"为什么？"

　　"因为我看到了他的眼睛。"沈柏寒说,"当他在和那几个孩子一起修机器人的时候,他的眼睛里有光。"

　　"那种光,我这辈子只见过两次。"

　　"一次是十年前,我刚开始写代码的时候,每次成功编译,我都会兴奋得睡不着觉。"
