　　【2026年8月20日，下午三点】

　　沈柏寒盯着屏幕，揉了揉发酸的眼睛。

　　连续五天，每天工作十六个小时，Gear的第一个MVP终于做出来了。

　　MVP——Minimum Viable Product，最小可行产品。

　　界面极其简陋，黑底白字，像九十年代的DOS系统。

　　没有漂亮的UI，没有流畅的动画，甚至连基本的美化都没有。

　　但核心功能跑通了。

　　用户可以填写问卷，系统可以生成齿轮画像，可以给出匹配建议。

　　虽然粗糙，但它能用。

　　沈柏寒点击”运行”，测试了第十遍。

　　一切正常。

　　他靠在椅背上，长长地呼出一口气。

　　然后拿起手机，拨通了一个号码。

　　“老张，是我，沈柏寒。”

　　“老沈？“电话那头的声音有些惊讶，“好久不联系了。怎么想起给我打电话？”

　　张建国，沈柏寒以前在上海一家互联网公司的同事。

　　两人一起做过几个项目，关系还不错。

　　后来张建国跳槽到了一家中型科技公司，现在是CTO。

　　“想请你帮个忙。“沈柏寒说，“我最近在做一个产品，想听听你的意见。”

　　“你在创业？“张建国笑了，“行啊，老沈。什么产品？”

　　“一个……项目管理工具。“沈柏寒说，“但和市面上的不太一样。”

　　“哦？“张建国来了兴趣，“有什么不一样？”

　　“这个……电话里说不清楚。“沈柏寒说，“你什么时候有空？我给你演示一下。”

　　“明天下午吧。“张建国说，“来我公司。正好我也想看看你在搞什么。”

　　“好，谢谢。”

　　挂断电话后，沈柏寒看着屏幕上那个简陋的界面，心里有些忐忑。

　　他知道张建国是个很务实的人，看问题很犀利。

　　如果连他都觉得Gear有价值，那就说明这个方向是对的。

　　如果他觉得不行……

　　沈柏寒摇摇头，不让自己往下想。

　　张建国的公司在陆家嘴，一栋甲级写字楼的二十三层。

　　沈柏寒站在楼下，仰头看着那栋玻璃幕墙的大楼，突然有种恍如隔世的感觉。

　　两年前，他也在这样的写字楼里工作。

　　西装革履，刷卡进门，坐在工位上敲代码。

　　那时候觉得很正常，很理所当然。

　　现在，站在这里，他感觉自己像个局外人。

　　电梯上到二十三楼，前台接待很客气地请他在休息区等候。

　　沈柏寒坐在真皮沙发上，看着四周——

　　落地窗外是黄浦江，对岸是东方明珠。

　　墙上挂着公司的发展历程、获奖证书、明星产品海报。

　　茶几上摆着最新一期的《财富》杂志。

　　一切都那么光鲜亮丽，那么成功。

　　沈柏寒低头看了看自己——

　　一件洗得有些发白的T恤，一条普通的牛仔裤，背着一个旧双肩包。

　　他突然有点后悔来这里。

　　“老沈！”

　　张建国从会议室里走出来，大步走过来，和沈柏寒握手。

　　四十岁出头，穿着得体的衬衫，戴着无框眼镜，脸上是成功人士特有的自信笑容。

　　“好久不见啊。“张建国拍了拍沈柏寒的肩膀，“瘦了不少。”

　　“是吗？“沈柏寒勉强笑了笑。

　　“来，我办公室聊。”

　　张建国的办公室很大，至少三十平米。

　　一张实木办公桌，两个书柜，一套会客沙发，还有一整面墙的落地窗。

　　“坐。“张建国示意沈柏寒坐在会客沙发上，自己泡了两杯茶，“说吧，什么产品？”

　　沈柏寒打开笔记本电脑，调出Gear的界面。

　　“是一个项目管理工具，“他说，“但它的逻辑和传统的不太一样。”

　　“传统的项目管理，是把任务分配给人，然后监控进度，追踪效率。”

　　“但Gear不是。”

　　“它是先了解每个人的特质——性格、技能、工作习惯。”

　　“然后根据这些特质，把任务匹配给最适合的人。”

　　“不是强迫人去做不适合的事，而是让每个人都能在自己擅长的领域发挥。”

　　张建国听着，脸上的笑容慢慢消失了。

　　“演示一下。“他说。

　　沈柏寒点开系统，开始演示。

　　首先是问卷——那些经过陆以辰修改的、更人性化的问题。

　　然后是齿轮画像——根据回答生成的可视化图形。

　　最后是任务匹配——系统自动推荐最适合的人选。

　　整个过程很流畅。

　　沈柏寒讲得很投入，因为他真的相信这个产品的价值。

　　但张建国的表情，越来越奇怪。

　　演示结束后，办公室里安静了几秒。

　　然后，张建国笑了。

　　不是赞赏的笑，而是一种带着怜悯的、无奈的笑。

　　“老沈，“他说，“你这是在做慈善，不是做SaaS。”

　　沈柏寒愣住了。

　　“什么意思？”

　　张建国站起来，走到窗边，点了根烟。

　　“你知道企业为什么买项目管理软件吗？“他转过身，“不是为了让员工舒服，是为了提升效率。”

　　“什么叫提升效率？就是让一个人干两个人的活，让员工不敢偷懒，让老板随时能知道谁在摸鱼。”

　　“你这个’寻找舒适区’的功能，“他指着屏幕，“哪个老板会买单？”

　　“老板恨不得员工都在非舒适区里拼命。”

　　“你让他们’只做擅长的事’？那不擅长的事谁做？”

　　“你让系统’尊重每个人的特质’？老板只想要一个标准化的执行机器。”

　　他走回来，坐在沈柏寒对面，语气变得更直白：

　　“老沈，你太理想主义了。”

　　“你把员工当齿轮，想让他们咬合；但资本把员工当电池，用完了就换。”

　　“你这款软件，是在帮电池延长寿命。”

　　“但老板只想要更大的电流。”

　　沈柏寒看着他，心里像被泼了一盆冷水。

　　“所以……“他的声音有些干涩，“你觉得这个产品没有市场？”

　　“不是没有市场。“张建国说，“是你找错了方向。”

　　“如果你把它改一改，加点功能，它还是能卖的。”

　　“什么功能？”

　　张建国打开自己的电脑，调出几款市面上主流的项目管理软件。

　　“你看，这些软件为什么卖得好？“他指着屏幕，“因为它们有这些功能——”

　　“屏幕监控：老板可以随时看到员工的电脑屏幕。”

　　“鼠标轨迹追踪：如果鼠标五分钟不动，自动标记为’摸鱼’。”

　　“工时强制统计：精确到分钟，加班时长一目了然。”

　　“邮件内容抓取：监控员工的邮件往来，防止泄密。”

　　“这些，才是企业愿意付钱的功能。”

　　他看着沈柏寒，语重心长地说：

　　“如果你把这些功能加到Gear里，再把那些’人文关怀’的部分藏起来，或者做成可选项。”

　　“我保证，它能卖出去。”

　　“而且卖得很好。”

　　沈柏寒看着那些功能列表，感觉到一种深深的恶心。

　　屏幕监控、鼠标追踪、工时统计……

　　这不是管理工具，这是囚笼。

　　“老沈，“张建国拍了拍他的肩膀，“我知道你有理想，但理想不能当饭吃。”

　　“你现在创业，压力肯定很大。”

　　“如果你不改，这东西在商业逻辑上是无法编译的。”

　　“它违背了资本的本性。”

　　“一文不值。”

　　沈柏寒走在街上，脑子里一片混乱。

　　张建国的话，像一把锤子，砸碎了他这些天建立起来的信心。

　　“无法编译。”

　　“一文不值。”

　　他想起了那个即将到期的租房合同。

　　想起了小宇每年15万的学费。

　　想起了陆以辰为了支持他，放弃了找工作的机会。

　　如果Gear真的卖不出去……

　　他们该怎么办？

　　沈柏寒站在路边，点了根烟。

　　手在微微发抖。

　　他拿出手机，打开银行APP。

　　账户余额：￥37,824

　　这是他们全部的积蓄了。

　　维持现在的生活，最多还能撑两个月。

　　两个月后，如果Gear还没有收入……

　　他不敢想下去。

　　沈柏寒深吸了一口烟，做了一个决定。

　　也许张建国是对的。

　　也许他真的太理想主义了。

　　也许，为了生存，他必须妥协。

　　凌晨一点，沈柏寒坐在电脑前。

　　他新建了一个文件夹：MonitorModule

　　监控模块。

　　然后开始写代码。

　　```python
　　class ScreenMonitor:
　　"""屏幕监控"""
　　def capture_screen(self, employee_id):
　　# 每5分钟截图一次
　　screenshot = take_screenshot()
　　upload_to_server(screenshot, employee_id)

　　class MouseTracker:
　　"""鼠标轨迹追踪"""
　　def track_activity(self, employee_id):
　　# 记录鼠标移动和点击
　　if idle_time > 300:  # 5分钟无操作
　　mark_as_idle(employee_id)

　　class WorktimeCounter:
　　"""强制工时统计"""
　　def count_hours(self, employee_id):
　　# 精确到秒的工时记录
　　login_time = get_login_time(employee_id)
　　logout_time = get_logout_time(employee_id)
　　return calculate_hours(login_time, logout_time)
　　```

　　每写一行，他的胃就翻涌一次。

　　这些代码，和他在大厂写的那些代码一模一样。

　　冰冷、高效、精准。

　　把人当作数据，当作可以被监控、被量化、被压榨的对象。

　　他想起了三年前，在公司被这些系统监控的日子。

　　每次离开工位超过十分钟，系统就会发警告。

　　每次下班早了，第二天就会收到HR的”关怀”。

　　那种被窥视的感觉，那种无处可逃的压迫感。

　　让他抑郁，让他崩溃，让他最终选择了逃离。

　　而现在，他正在亲手制造同样的东西。

　　沈柏寒的手指停在键盘上，久久没有动。

　　他看着屏幕上那些代码，突然有种想吐的冲动。

　　“柏寒？”

　　陆以辰的声音在身后响起。

　　沈柏寒猛地关掉屏幕，转过身：“你怎么还没睡？”

　　“听见你在敲键盘。“陆以辰走过来，“在写什么？”

　　“没什么，“沈柏寒说，“就是……优化一些功能。”

　　陆以辰看了看他，然后看向电脑。

　　“能给我看看吗？”

　　“不用了，“沈柏寒说，“太技术化了，你看不懂。”

　　“那就用你看得懂的方式给我讲讲。“陆以辰坐在旁边，“你在加什么功能？”

　　沈柏寒沉默了。

　　陆以辰等着。

　　良久，沈柏寒重新打开屏幕。

　　陆以辰看见了那些代码，看见了文件夹的名字：MonitorModule。

　　虽然他不懂代码，但他认识那个词。

　　Monitor。

　　监控。

　　“这是……“陆以辰的声音有些紧绷，“监控模块？”

　　沈柏寒点点头。

　　“为什么要加这个？”

　　“因为……“沈柏寒深吸一口气，“因为张建国说，不加这些，Gear卖不出去。”

　　“企业需要的不是理解员工，而是监控员工。”

　　“如果我们想赚钱，就必须做他们想要的东西。”

　　陆以辰看着他，眼神里有一种他从未见过的失望。

　　“所以你就打算做？”

　　“我……“沈柏寒说不出话。

　　“柏寒，“陆以辰的声音很平静，但每个字都像锤子，“这跟之前的Gear不一样了。”

　　“我知道。“沈柏寒说，“但我们需要钱，以辰。”

　　“我们的积蓄只够撑两个月了。”

　　“小宇的学费，房租，生活费……”

　　“如果Gear卖不出去，我们该怎么办？”

　　“那你当初为什么要辞职？“陆以辰打断他，“你在大厂，做的不也是这些东西吗？”

　　“监控员工，压榨员工，把人当作工具。”

　　“你不是因为受不了才离开的吗？”

　　“现在你要重新做回去？”

　　“那这两个月，我们在干什么？”

　　“那些关于齿轮、关于理解、关于让每个人找到自己位置的理念，都是假的吗？”

　　沈柏寒被问得哑口无言。

　　陆以辰站起来，走到窗边。

　　外面是深夜的上海，万家灯火，璀璨夺目。

　　“我不是不理解你的压力。“他说，背对着沈柏寒，“我知道你在为这个家承担很多。”

　　“我知道我没有收入，是个负担。”

　　“但柏寒，“他转过身，眼眶发红，“如果我们为了钱做这个，那你为什么要创业？”

　　“你回大厂赚钱不是更快吗？”

　　“你在那里，一年几十万，稳定，体面。”

　　“为什么要冒这么大的风险，搞这么一个可能失败的东西？”

　　沈柏寒看着他，突然说不出话。

　　对啊。

　　为什么？

　　如果只是为了钱，他有太多更容易的选择。

　　“因为你想做不一样的东西。“陆以辰说，“因为你受够了那种把人当工具的世界。”

　　“因为你想证明，软件也可以是温暖的，是尊重人的。”

　　“但现在，你要亲手毁掉这些。”

　　“别写这种代码，柏寒。”

　　“别写那种……连你自己都不想用的代码。”

　　“如果我们输了，大不了去送外卖，去搬砖。”

　　“但不要变成我们讨厌的那种人。”

　　就在这时，小宇从卧室里走出来。

　　他揉着眼睛，显然是被吵醒了。

　　“爸爸？陆爸爸？“他走过来，“你们在吵架吗？”

　　“没有，“沈柏寒说，“我们只是在讨论工作。”

　　小宇走到电脑前，看了一眼屏幕。

　　屏幕上是沈柏寒刚才写的监控模块的界面演示图——

　　一个巨大的红色圆圈，中间是一只眼睛的图标。

　　下面是实时的员工屏幕截图，还有鼠标轨迹的热力图。

　　整个界面充满了冰冷的、机械的、压迫感。

　　小宇看着那个红色的圆圈，本能地后退了一步。

　　“爸爸，“他的声音有些颤抖，“这个圆圈看起来像一只红色的眼睛。”

　　“很吓人。”

　　沈柏寒的心脏像被什么东西狠狠击中了。

　　小宇的直觉，永远是最准确的。

　　那不是一个功能图标。

　　那是一只眼睛。

　　一只冰冷的、无处不在的、无情的眼睛。

　　在监视着，在窥探着，在评判着。

　　就像《1984》里的老大哥。

　　就像他曾经在大厂感受到的那种窒息。

　　“小宇，“沈柏寒蹲下来，和他平视，“你害怕这个吗？”

　　小宇点点头：“它在看我。”

　　“它一直在看我。”

　　“我做什么都被它看见。”

　　“我想躲起来，但躲不掉。”

　　沈柏寒抱住小宇，感觉到他在微微发抖。

　　那一刻，他突然明白了。

　　这个监控系统，对小宇来说，就是最大的恐惧。

　　小宇需要的是独处，是自己的空间，是不被打扰的专注时光。

　　而监控，剥夺了这一切。

　　它把人置于永恒的注视之下，让人无处可逃。

　　这不是管理。

　　这是暴力。

　　沈柏寒松开小宇，转身走到电脑前。

　　他的手放在鼠标上，悬停了几秒。

　　然后，他选中了整个MonitorModule文件夹。

　　按下Delete。

　　“确定要删除吗？“系统弹出提示框。

　　沈柏寒毫不犹豫地点击”确定”。

　　那些代码，那些他花了一整晚写的代码，

　　消失了。

　　他关掉电脑，转身看着陆以辰和小宇。

　　“对不起。“他说，声音有些哽咽，“我差点就……”

　　“没事。“陆以辰走过来，抱住他，“你醒过来了。”

　　小宇也走过来，抱住沈柏寒的腿。

　　三个人就这样抱在一起，在深夜的书房里。

　　窗外，城市还在运转。

　　那些监控系统，那些冰冷的算法，还在压榨着无数个像他们一样的人。

　　但至少，在这个小小的房间里，

　　他们拒绝了。

　　第二天早上，沈柏寒醒来的时候，发现陆以辰已经在客厅了。

　　他面前摊着一张纸，上面写满了字。

　　“你在写什么？“沈柏寒问。

　　“在想，“陆以辰说，“既然主流市场不接受Gear，那我们应该去找非主流市场。”

　　“什么意思？”

　　“你看，“陆以辰指着纸上的列表，“哪些地方不追求利润最大化，而追求人的成长？”

　　“学校。”

　　“哪些地方不把人当工具，而把人当目的？”

　　“公益组织。”

　　“哪些地方理解’每个人都是独特齿轮’这个理念？”

　　“那些为特殊人群服务的机构。”

　　沈柏寒看着那张纸，眼睛慢慢亮了起来。

　　“你的意思是……”

　　“我们不做企业市场。“陆以辰说，“我们做教育市场。”

　　“第一个客户，就是源码实验学校。”

　　“叶青校长一定会理解Gear的价值。”

　　“因为他本来就在做同样的事——为每个孩子找到合适的位置。”

　　沈柏寒愣了几秒，然后突然笑了。

　　对啊。

　　为什么一定要卖给企业？

　　为什么一定要迎合资本的逻辑？

　　Gear的理念，本来就不是为了压榨人，而是为了理解人。

　　那么，它应该去那些真正需要理解人的地方。

　　“以辰，“沈柏寒说，“你是天才。”

　　“我不是天才，“陆以辰笑了，“我只是个文科生。”

　　“但至少，我知道什么是对的。”

　　两人相视一笑。

　　在这个清晨，在这间简陋的出租屋里，

　　他们找到了新的方向。

　　不是向资本妥协，不是向市场屈服，

　　而是去寻找那些，真正需要Gear的人。

　　那些相信人不是工具，而是独特存在的人。

　　那些愿意为每个齿轮，找到合适位置的人。

　　那天晚上，沈柏寒在日志里写道：

　　【寄养日志·Day 44】

　　昨天，我差点毁掉Gear。

　　我去见了一个以前的同事，想听他对产品的意见。

　　他说，Gear在商业逻辑上是”无法编译的错误”。

　　因为它违背了资本的本性。

　　他建议我加入监控功能——屏幕监控、鼠标追踪、工时统计。

　　那些把人当作电池，用完就扔的功能。

　　我动摇了。

　　因为账户里只剩三万多块钱。

　　因为还有两个月就要搬家。

　　因为我不知道Gear能不能卖出去。

　　于是我写了那些代码。

　　那些我曾经最讨厌的、最恶心的代码。

　　但以辰阻止了我。

　　他说：如果只是为了钱，你为什么要创业？

　　他说：别写那种连你自己都不想用的代码。

　　然后小宇看见了监控界面上的红色眼睛。

　　他说：很吓人。

　　那一刻，我突然清醒了。

　　我在做什么？

　　我为了逃离监控系统而辞职。

　　现在却要亲手制造一个更强大的监控系统？

　　我删掉了那些代码。

　　全部删掉。

　　一行都不留。

　　然后陆以辰给了我一个新的方向——

　　既然企业不需要Gear，那就去找真正需要它的地方。

　　学校。

　　公益组织。

　　那些把人当人看的地方。

　　今天早上，我给叶青校长发了邮件。

　　说我们有一个系统，可以帮助学校更好地理解每个学生。

　　不是为了监控他们，而是为了找到适合他们的教学方式。

　　不知道他会不会回复。

　　但至少，我们又开始转动了。

　　这次，转向了正确的方向。

　　张建国说，Gear在商业逻辑上无法编译。

　　也许他是对的。

　　但谁说我们一定要符合商业逻辑？

　　谁说软件一定要为资本服务？

　　我们可以为教育服务，为人性服务，为那些被忽视的齿轮服务。

　　这条路可能更难，可能更穷，可能最后一分钱都赚不到。

　　但至少，我们不会变成自己讨厌的人。

　　至少，我们还能在镜子里，看见自己的眼睛。

　　而不是那只红色的、冰冷的、无处不在的眼睛。
