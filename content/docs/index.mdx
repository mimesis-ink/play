# 两种爱的语言

一部关于爱、疾病与时代的现代都市小说。

## 故事简介

这是一个关于两个截然不同的人——哲学研究生陆以辰和程序员沈柏寒——在时代洪流中相遇、分离、重逢的故事。

**陆以辰**：追求"绝对本质"的孤独观察者，哲学系研究生，用理性试图解构一切，包括爱本身。

**沈柏寒**：渴望"确定性反馈"的迷途工匠，程序员，在代码的确定性中寻找安全感，却在现实中屡屡碰壁。

2015年，在上海弄堂深处的一家小维修店，十七岁的陆以辰与三十一岁的沈柏寒有过一次短暂的相遇。那只是两个平行世界的唯一交点。

2024年平安夜，失业、离婚、疾病——三重打击下的沈柏寒，在绝望中给九年前那个少年发了一条消息："我离婚了。"

陆以辰回复："记得。"

故事，正式开始。

## 作品结构

小说分为三个部分：

### 主线章程（第2-59章）
主要故事线，涵盖：
- **现在的切片（第2-40章）**：从2024年12月的重逢，到2026年春节的新生活。讲述两人如何在疾病、偏见、时代冲击中建立起一个微型避难所。
- **未来的重构（第41-59章）**：讲述两人带着历史的伤痕，走向属于他们的结局。

### 时代卷：时代洪流（插入章节）
历史的洪流（宏观、群像、厚重）。以编年史的形式，回溯2015-2024年的十年变迁，展现两个人如何在时代的绞肉机里各自挣扎。

## 核心主题

- **非对称的爱**：年龄差（16岁）、健康差异、思维方式的差异，如何在不对等中建立真正的连接
- **时代的幸存者**：从"双创"泡沫到疫情封控，从P2P爆雷到AI冲击，两个普通人如何在历史洪流中求生
- **具体的胜利**：对抗宏大理论的暴政，捍卫"此时此地"的真实经验
