# 两种爱的语言 - 文档结构说明

## 概述

本项目是基于 Fumadocs 构建的文档网站，用于展示小说《两种爱的语言》的内容。

## 目录结构

### 第一卷：崩塌与共生（章节 1-40）
`/docs/part1/`

**时间线**：2024年12月 - 2026年春节

**主要内容**：
- 第1-21章：连接与错位（相遇、重逢、试探）
- 第22章：诊断（确诊 HIV）
- 第23章：空白的房间（确立关系）
- 第24-25章：过去（告知前妻）
- 第26-28章：审判（母亲、老张、林教授的质疑）
- 第29-30章：内爆（寻魂App事件、小K）
- 第31章：见证（前任林晓到访）
- 第32章：体温（U=U，肉体结合）
- 第33-40章：日常生活、职业冲击、搬家、除夕回望

### 回溯卷：时代洪流
`/docs/timeline/`

**时间线**：2015-2024

编年史形式展现十年社会变迁：
- 2015-2017：泡沫与迷途（双创热潮）
- 2018-2019：寒冬与裂痕（P2P爆雷）
- 2020-2022：静默与孤岛（疫情三年）
- 2023-2024：崩塌与幸存（AI冲击、裁员）

### 第三卷：非对称的爱（章节 41-59）

**主要内容**：
- 第41章：归来（带着历史生活）
- 第42章：复学（陆以辰重返校园）
- 第43章：答辩（论文《非对称的爱》）

- 第46章及后续：结局与延伸

### 配角群像

### 附录
`/docs/appendix/`

- `outline.mdx` - 创作大纲

## 核心主题

1. **非对称的爱**：年龄差（16岁）、健康差异、思维方式的差异
2. **时代的幸存者**：从"双创"到疫情，从P2P到AI，两个普通人的求生
3. **具体的胜利**：对抗宏大理论，捍卫"此时此地"的真实经验

## 技术栈

- **框架**：Next.js 16.1.1
- **文档系统**：Fumadocs (fumadocs-core 16.4.7)
- **内容格式**：MDX
- **样式**：Tailwind CSS 4.1.18

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问文档
# http://localhost:3000/docs
```

## 内容格式

所有章节采用 MDX 格式，包含 YAML frontmatter：

```mdx
---
title: "章节标题"
---

章节内容...
```

## 导航结构

文档使用 `meta.json` 文件控制导航顺序和结构：

- `content/docs/meta.json` - 根导航
- `content/docs/part1/meta.json` - 第一卷章节列表
- `content/docs/timeline/meta.json` - 回溯卷章节列表
- `content/docs/part3/meta.json` - 第三卷章节列表
- 其他各部分的 `meta.json`

## 特色功能

- ✅ 清晰的章节导航

- ✅ 自动生成的目录（TOC）
- ✅ 全文搜索支持
- ✅ 响应式设计
